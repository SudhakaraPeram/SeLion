JSON DataProvider

JSON DataProvider is an implementation of SeLion DataProvider that operates on JSON files

To find out more about the JSON file format please refer to Getting started with JSON. JSON provides a convenient alternative to XML files. JSON is easy, for humans to read and write, also for machines to parse.
