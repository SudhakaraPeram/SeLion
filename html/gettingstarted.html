Browser Automation

How to create a browser based test using SeLion.

Prerequisites

Make sure the following software is installed on your machine before you start using SeLion:

  1. Java JDK 1.7
  2. Maven 3.2.x
  3. A web browser. This example uses Firefox <= version 32
